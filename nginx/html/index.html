GP Connect demonstrator

The GP Connect demonstrator is:

  • a provider system reference implementation of the GP Connect API
  • a consumer system demonstrator built against the reference implementation
  • interactive API documentation for developers to test calls to the reference implementation

Note: GP Connect is an FHIR® based API for communicating with GP IT systems. For more information on GP Connect please see the GitHub development portal.

Consumer system demonstrator

The consumer system demonstrator illustrates how the GP Connect API might be integrated with an electronic patient record (EPR) or Trust portal system.


Consumer system demonstrator screenshot

API reference implementation

This is a reference implementation of the GP Connect API in order to develop and test prospective GP Connect consumer systems, and to provide data for the Consumer system demonstrator and interactive documentation.

To interoperate with the reference implementation using your own development/test environment, use the following FHIR base URLs:

GP Connect 0.x (DSTU2) https://orange.testlab.nhs.uk/gpconnect-demonstrator/v0/fhir
GP Connect 1.x (STU3) https://orange.testlab.nhs.uk/gpconnect-demonstrator/v1/fhir
